# Asset Log

月ごとの資産状況を手入力で記録し、推移・配分を可視化する個人向け資産管理アプリ。

## 目的

資産の月次スナップショットを手入力で蓄積し、時系列の推移やカテゴリ別配分をグラフで確認できるシンプルな個人向けアプリ。

## v1 スコープ

- Magic Link ログイン
- 月次スナップショット登録
- 資産カテゴリ別入力
- 前月比表示
- 推移グラフ
- 月次メモ
- 編集・削除

## 技術スタック

| 用途 | 技術 |
|------|------|
| フレームワーク | Next.js (App Router) |
| 言語 | TypeScript |
| スタイリング | Tailwind CSS |
| バックエンド / 認証 | Supabase |
| グラフ | Recharts |
| ホスティング | Vercel |
